# Agent-Learning

一个用于学习和实践 LangChain / LLM 应用开发的示例仓库，内容以 Jupyter Notebook、Markdown 笔记和少量 Python 脚本为主。

## 项目内容

当前仓库主要包含以下部分：

- `chapter-01-summary/`
  - `helloworld.ipynb`
  - `test.py`
- `chapter-02-model-io/`
  - 模型调用、提示词模板、示例选择器等 Notebook / Markdown / Python 示例
- `langchain/`
  - LangChain 学习 PDF 资料

## 适用场景

这个仓库适合用于：

- 学习 LangChain 基础用法
- 练习模型调用（LLM / Chat Model / Embeddings）
- 了解 Prompt Template、Few-shot、Example Selector 等常见组件
- 在 Notebook 中快速验证 LLM 代码

---

## 环境要求

建议使用：

- Python 3.10 或 3.11
- pip 最新版本
